# PDF 蓋章器

一個可以幫你在 PDF 檔案上蓋章的小工具

> 前置條件：需要有去背章，或者背景白底
> 目前限制：要調整幾個設定，而且章沒有辦法調大小

1. 圖片名稱（變數名：STAMP_IMG）
2. 輸入的檔案名稱（變數名：PDF_INPUT）
3. 輸出的檔案名稱（變數名：PDF_OUTPUT）
4. PDF 第幾頁（變數名：PAGE_NUM）
5. 大小章的切割，程式根據你的切點比例判讀公司大章與小章（變數名：SPLIT_RATIO）

## 如何使用（Python 環境）

> 請安裝 Python（現環境為 Win11，Python 3.12）

```shell
pip install -r requirements.txt
python stamp_pdf.py
```

## 預期改善

1. 可以下參數，預期以下幾個
  - 輸入的 PDF
  - 輸入的圖片
  - 輸出的 PDF（預設：輸入的 PDF 檔名加上 `_已用印` ）
2. 視窗處理
  - 第 1. 點如果預期參數如果沒有下好，開一個視窗給使用者去填寫下列參數（要可以選擇檔案）
    - 輸入的 PDF
    - 輸入的圖片
    - 要匯入的圖檔
  - 滑鼠個別縮放章的大小
3. 未想好的需求
  - exe 打包 (Nuitka)
  - 多頁面貼上（一個裡面有好幾頁都要蓋章）
